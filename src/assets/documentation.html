IntelliDesk Help Documentation

Keyboard shortcuts and settings guide for IntelliDesk application

Keyboard Shortcuts

Here are the keyboard shortcuts available in IntelliDesk:

Shortcut Action
CTRL+P / CTRL+p Open Conversations Panel
CTRL+S / CTRL+s Open Settings
ESC Close Active Modal (Settings, File Uploads)
ALT+A / ALT+a Toggle Autoscroll
CTRL+F / CTRL+f Attach Files
CTRL+N / CTRL+n New Conversation
CTRL + = Zoom In (Increase Font Size)
CTRL + - Zoom Out (Decrease Font Size)
CTRL + 0 Reset Zoom (Reset Font Size)
CTRL+D / CTRL+d Show this documentation

Settings

Available settings in IntelliDesk:

Setting Description Default
Mode/Theme Switch between Dark and Light Mode System Preference
Auto Scroll on Output for AI Automatically scroll to the latest output from the AI ON
Language Select the language for the interface English

Conversations Panel/Chats Help

Right-click on a Conversation

Right-clicking on a conversation in the panel will show you options to:

  • Delete the conversation
  • Rename the conversation
  • Pin important conversations
  • Export conversation as text file

Examples

Here are some examples of how to use the shortcuts and settings:

Open Conversations Panel

To open the conversations panel, press CTRL+P or CTRL+p.

Open Settings

To open the settings, press CTRL+S or CTRL+s.

Close Active Modal

To close the active modal (such as settings or file uploads), press ESC.

Zoom Controls

To adjust the zoom level of the content area:

  • Zoom in: CTRL + =
  • Zoom out: CTRL + -
  • Reset zoom: CTRL + 0

Toggle Auto Scroll

To toggle auto-scroll, press ALT+A or navigate to the settings and switch the Auto Scroll on Output for AI option.

Attach Files

To quickly attach files to your conversation, press CTRL+F.

Right-click on Conversation

To manage conversations, right-click on a conversation in the panel and select options like Delete or Rename.
